# Sumanth Pasupuleti Portfolio

A high-fidelity, interactive personal portfolio website showcasing expertise in **Data Engineering**, **Cloud Architecture**, and **IT Solutions**.

## 🚀 Features
- **Dynamic Hero Section**: Interactive DotField background and BorderGlow profile photo effects.
- **Project Gallery**: Immersive grid featuring auto-playing video backgrounds for real-time projects.
- **Professional Experience**: High-contrast, beautifully styled sections for career history with custom UI mockups.
- **Modern Tech Stack**: Built with Vanilla HTML5, CSS3 (Tailwind CSS), and custom JavaScript for performance and visual excellence.
- **Responsive Design**: Fully optimized for all screen sizes, from mobile to ultra-wide displays.

## 📂 Project Structure
```text
├── assets/
│   ├── images/         # Project mockups and profile photos
│   └── videos/         # Animated background and project showcase videos
├── css/
│   └── BorderGlow.css  # Custom styling for interactive components
├── js/
│   ├── border-glow.js  # Logic for profile image glow effect
│   └── dot-field.js    # Logic for animated hero background
├── index.html          # Main entry point
└── README.md           # Project documentation
```

## 🛠️ Setup
Simply open `index.html` in any modern web browser to view the portfolio.
